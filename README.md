## 🚀 **Quick Start**

```bash
# 🔥 Get started in 30 seconds
git clone https://github.com/exivox/frontend.git
cd exivox
npm install

# 🌟 Launch the future
cp .env.example .env
npm run dev

# 🎉 Open http://localhost:5173
```
